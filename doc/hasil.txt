hai, ini adalah command khusus untuk admin grup.\r\n\r\n- /set_welcome: untuk mengatur pesan sambutan\r\n- /test_welcome: untuk melihat hasil contoh pesan sambutan\r\n- /set_goodbye: digunakan untuk mengatur pesan selamat tinggal ketika ada pengguna yang mengeluarkan diri\r\n- /set_welcome_mode: digunakan untuk menyalakan atau mematikan pesan selamat datang (ini berguna jika sudah ada bot lain yang mengucapkan selamat datang. biar ga bentrok aja)\r\n- /set_goodbye_mode: ini hampir sama dengan welcome, hanya aja ini digunakan untuk mematikan dan menyalakan pesan selamat tinggal ketika ada user keluar grup
