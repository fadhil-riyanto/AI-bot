hai, ini adalah command khusus untuk admin grup.

- /set_welcome: untuk mengatur pesan sambutan
- /test_welcome: untuk melihat hasil contoh pesan sambutan
- /set_goodbye: digunakan untuk mengatur pesan selamat tinggal ketika ada pengguna yang mengeluarkan diri
- /set_welcome_mode: digunakan untuk menyalakan atau mematikan pesan selamat datang (ini berguna jika sudah ada bot lain yang mengucapkan selamat datang. biar ga bentrok aja)
- /set_goodbye_mode: ini hampir sama dengan welcome, hanya aja ini digunakan untuk mematikan dan menyalakan pesan selamat tinggal ketika ada user keluar grup
