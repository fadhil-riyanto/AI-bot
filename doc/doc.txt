Note digunakan untuk menyimpan dan melihat catatan
ini berguna untuk menyimpan teks kamu

/note {nama} {note kamu}: digunakan untuk mensave note kamu
/my_note: digunakan untuk melihat isi list note kamu
/get_note {nama}: digunakan untuk melihat note kamu
/del_note {nama}: digunakan untuk menghapus note
/clear_note: hapus semua note (tidak bisa dikembalikan)
