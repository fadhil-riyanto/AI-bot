ini adalah list tool tool yang bot ini miliki

- /wiki: dapatkan isi wikipedia
- /azan: cek jadwal azan.
- /tanggal: lupa tanggal? tenang...
- /tr: ini bisa untuk translate
- /tulis: digunakan untuk menulis buku
- /waktu: cek jam yuk
- /berita: agar update yuk baca berita
- /br: cari jawaban di brainly
- /capture: untuk screenshot halaman web
- /corona: info corona terkini
- /corona_provinsi: info corona di tiap tiap provinsi
- /cuaca: cek cuaca
- /gempa: cek gempa
- /q: buat quotes kamu jadi gambar!
- /quran:  kitab quran sekarang ada di telegram (aliases /qs)
- /short: pendekkan link biar rapi
- /xlsx_to_json: convert xlsx excel ke format json
- /resi: cek resi
- /upload: reply gambarmu untuk diupload ke web (aliases /ui)
- /upload_sticker: reply stickermu untuk diupload ke web (aliases /us)
- /ts: convert image ke stiker
- /tf: convert stiker ke image
- /reverse: reply dengan gambar untuk mencari data di google, bing (aliases /i_rev)
- /s_rev: reverse stiker
- /composer {package}: untuk mencari info di composer package
- /calculator: hitung kalkulator secara inline keyboard! (aliases /c)
